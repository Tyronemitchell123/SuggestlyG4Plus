## üöÄ Getting Started

1. **Clone the Repository**

   ```bash
   git clone https://github.com/suggestly/elite-platform.git
   cd elite-platform
   ```

2. **Install Dependencies**

   ```bash
   pnpm install
   ```

3. **Configure Environment**

   ```bash
   cp .env.example .env
   # Edit .env with your API keys
   ```

4. **Start Development Server**

   ```bash
   pnpm dev
   ```

5. **Build for Production**
   ```bash
   pnpm build
   pnpm preview
   ```
